# <a name="xamarinforms-local-data-storage"></a>Xamarin.Forms lokale Speicherung von Daten

## <a name="filesfilesmd"></a>[Dateien](files.md)

Die Dateiverarbeitung mit Xamarin.Forms kann mit Code in einer .NET Standard-Bibliothek oder mithilfe eingebetteter Ressourcen erfolgen. In diesem Artikel wird erläutert, wie zum Verarbeiten von freigegebenen Code in einer Xamarin.Forms-Anwendung ausführen wird.

## <a name="local-databasesdatabasesmd"></a>[Lokale Datenbanken](databases.md)

Xamarin.Forms unterstützt datenbankgesteuerte Anwendungen über die SQLite-Datenbank-Engine. Dadurch ist es möglich, Objekte in freigegebenem Code zu laden und zu speichern. In diesem Artikel wird beschrieben, wie Xamarin.Forms-Apps Daten mithilfe von SQLite.Net in eine lokale SQLite-Datenbank lesen und schreiben können.
